\# HR Analytics using SQL



This project focuses on analyzing employee attrition data using SQL.



The main objective of this project is to identify

attrition trends based on department and gender.



\## Tools Used

\- DB Browser for SQLite

\- SQL

\- SQLite Database



\## Dataset

The dataset contains employee information such as:

Age, Gender, Department, Attrition, Job Role, Monthly Income, etc.



Table name used in the project:

\- hr\_attrition



\## Analysis Performed



1\. Overall employee count and attrition rate

2\. Department-wise attrition analysis

3\. Gender-wise attrition analysis



\## SQL File

\- hr\_analysis.sql



\## Database File

\- hr\_analytics.db
